# 운동 기록앱,  FitnessApp

## ● 프로젝트 개요
 > 개인 운동 목표와 진행 상황을 기록할 수 있는 애플리케이션으로 SwiftUI 학습을 위해 개발하였습니다. <br> 이때, MVVM 방식의 상태관리 모듈화에 집중하여 개발을 진행했습니다.
